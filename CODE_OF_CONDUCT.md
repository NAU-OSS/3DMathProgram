# Expected behavior for community members
Treat people how you want to be treated and make sure you are making a positive impact on the community.

# Unacceptable behavior examples
Saying curse words in code review and personal attacks on people because of their code.

# How to report violations
Contact a mod.

# Enforcement guidelines
Banishment from contributing if violations are common.

# Community values and principles

- Be kind
- Do great things
- Make positive contributions
