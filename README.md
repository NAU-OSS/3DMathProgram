# 3DMathProgram

3D math program is a pre built software that does 3D math operations for users. This is used mostly for thinks like raycasting and ray tracing. This project is designed to be used for 3D math operations and gives the user a pre built library for doing 3D math. 


# Features and usage examples

- Normalization
- reflection
- length
- angle
- subtraction
- addition
- cross product
- dot product

# Local installation
git clone https://github.com/NAU-OSS/3DMathProgram
cd 3DMathProgram

Now you can play with this program yourself!
